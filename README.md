# :trumpet: cspot_vita
A Spotify player for PS Vita.

*Only to be used with premium spotify accounts!*

*Requires [iTLS-Enso](https://github.com/SKGleba/iTLS-Enso) or newer firmware.*

## Building

### Prerequisites
- [vitasdk](https://github.com/vitasdk)

```shell
mkdir build && cd build
cmake .. && make
```
